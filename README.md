# QCM Point Counter

Application web statique pour calculer une note sur 20 à partir des réponses à un QCM. Deux modes de correction
disponibles : **correction immédiate** et **correction globale**.

## 🎯 Fonctionnalités

### Mode Correction Immédiate

- Voyez votre score **après chaque question**
- Retour instantané sur la bonne ou mauvaise réponse
- Accumulation des points question par question
- Note finale calculée sur 20

### Mode Correction Globale

- Répondez à **toutes les questions** avant la correction
- Correction complète à la fin
- Vue d'ensemble de vos résultats
- Export du récapitulatif en PDF
